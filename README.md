# Smartकृषि 🌾

Smartकृषि is an AI-powered crop recommendation and farming advisory system developed using Machine Learning, Flask API, React.js, and scikit-learn.

The system recommends suitable crops based on soil nutrients and environmental conditions while also providing personalized farming guidance.

---

## 🚀 Features

- AI-based crop recommendation
- Personalized crop ranking
- Live weather integration using Open-Meteo API
- Irrigation guidance
- Fertilizer recommendation
- Pest control advisory
- Weather guidance
- Disease detection demo module

---

## 🛠 Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Flask
- REST API

### Machine Learning
- scikit-learn
- Pandas
- NumPy

### API
- Open-Meteo API

---

## 📊 Input Parameters

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Humidity
- Rainfall
- Farm Size
- Previous Crop
- Season

---

## 🎯 Output

- Recommended crops
- Personalized ranking
- Farming advisory
- Weather insights
